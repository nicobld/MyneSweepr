# MyneSweepr
An epic MineSweeper game made in C++ with the SDL2 library.

## Requirements
_-> SDL2_  
You can install it with Homebrew  
`brew install SDL2`  

## Run
Start by compiling if not compiled  

`make`  

Then to run the game either do  

`./prog "easy/medium"`  

or  

`./prog <width> <height> <bombs>`
