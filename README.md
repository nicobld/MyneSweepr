# MyneSweepr
Multiplayer minesweeper game made in C++ with the SDL2 library.  

## Requirements
_SDL2_  

## Run
Start by compiling if not compiled  

`./build.sh`  

To run the server

`./bin/server_MyneSweepr`  

To run the client

`./bin/client_MyneSweepr`
