# MyneSweepr
MineSweeper game made in C++ with the SDL2 library.  

## Requirements
_SDL2_  

## Run
Start by compiling if not compiled  

`make`  

Then to run the game either do  

`./prog "easy/medium"`  

or  

`./prog <width> <height> <bombs>`
